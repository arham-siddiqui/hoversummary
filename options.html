Connect to Ollama

Hover Summary now runs the vision model locally on your Mac. Your captured region stays on your computer and no paid API key is required.

The model must be downloaded in Ollama before it can be used.

One-time Mac setup

Install and open Ollama, then run ollama pull qwen3-vl:4b-instruct. To let the extension connect, run launchctl setenv OLLAMA_ORIGINS "chrome-extension://*", then completely quit and reopen Ollama.
